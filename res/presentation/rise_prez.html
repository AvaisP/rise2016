Problem Statement Simplified:

  • Corporate Treasurers and stakeholders in the banking ecosystem face two primary issues
  • Data Visualization
  • Predictive Analytics

Our solution encompasses the following aspects:

  • Visualization of Account and Transaction Data
  • Predictive Algorithms to analyse credit risk trade-offs
  • Simplified Analytics based on classification from real-time datasets

Visualization

  • History of minimum balances across a given timeline
  • Naive Bayes Algorithm trained to estimate futures
  • Notification-based system to avoid potential losses

Predictions

  • Assessing currency patterns via logistic regression
  • Offering credit risk analysis for the investment portfolio

Analytics

  • Automated analysis of potential transfers across accounts
  • Classifying transactions as essential and non-essential

Technologies and API Usage

Our webapp is built on Django using a number of Javascript libraries and Python Packages for machine learning such as scikit-learn, scipy and numpy

  • Barclays API to pull customer Account data
  • Yodlee API for user data, transactions and investments
  • Twilio API for SMS and calling services
  • IBM Watson Tradeoff Analytics
